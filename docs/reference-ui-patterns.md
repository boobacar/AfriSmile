# Analyse UI — sites de référence (inspiration)

Références observées: ultradent.com, tdsc.com, skydentalsupply.com/anesthetics, benco.com, dentaltix.com/fr, henryschein.fr.

## Patterns communs retenus

1. **Header professionnel multi-niveaux**
   - Topbar informationnelle (livraison, hotline, compte pro)
   - Navbar principale claire avec CTA visible (devis / contact)
   - Navigation desktop dense + menu mobile simple

2. **Hero orienté conversion**
   - Message de valeur fort + sous-texte concret
   - CTA primaire (catalogue / devis) + CTA secondaire (PDF / contact)
   - Présence d’éléments de confiance (badges service/SAV/livraison)

3. **Catalogue produit lisible**
   - Cartes avec hiérarchie nette: nom, prix/mention, avantages, actions
   - Badges de statut et boutons d’action directs
   - Filtres/catégories accessibles (souvent sticky)

4. **Blocs de réassurance**
   - Icônes/badges: support, délais, expertise, installation
   - Sections dédiées aux services techniques et offres pro

5. **Système visuel cohérent**
   - Typographie forte pour les titres
   - Espacement généreux, cartes arrondies, ombres douces
   - Couleurs d’accent cohérentes pour CTA et informations importantes

6. **Promotions et CTA persistants**
   - Bandeau promo clair
   - Actions flottantes de contact (WhatsApp / devis)

7. **Footer complet**
   - Navigation secondaire, contact, mentions légales, CTA téléchargement

## Application sur AfriSmile

- Conserver le contenu métier, le branding existant et le logo local.
- Moderniser la présentation: composants uniformes (buttons/cards/chips/forms), meilleur responsive, CTA mieux distribués.
- Renforcer les parcours B2B: devis, PDF, service technique, espace pro.
