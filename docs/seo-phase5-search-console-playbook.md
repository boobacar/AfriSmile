# SEO Phase 5 — Search Console Domination Playbook (AfriSmile)

## Objectif
Augmenter les clics organiques qualifiés Afrique (Sénégal + pays cibles) en optimisant CTR, positions 4-20, et conversion vers `/contact`.

## 1) Requêtes prioritaires (cluster business)

### Cluster A — Transactionnel direct (priorité haute)
- matériel dentaire sénégal
- fournisseur matériel dentaire sénégal
- devis matériel dentaire dakar
- fauteuil dentaire sénégal prix
- autoclave classe b sénégal
- scanner intra oral sénégal

### Cluster B — Expansion Afrique (priorité haute)
- matériel dentaire côte d’ivoire
- matériel dentaire abidjan
- matériel dentaire cameroun
- équipement dentaire douala
- matériel dentaire ghana
- matériel dentaire nigeria

### Cluster C — Intention comparatif / ROI (priorité moyenne)
- prix matériel dentaire dakar
- devis cabinet dentaire abidjan
- scanner intra oral accra prix
- fournisseur matériel dentaire lagos
- ouvrir cabinet dentaire afrique de l’ouest

## 2) Cadence hebdomadaire (exécution)

### Tous les lundis (45 min)
1. Ouvrir Search Console > Performance > 7 derniers jours.
2. Filtrer: pages contenant `/blog/` puis `/materiel-dentaire-`.
3. Extraire:
   - requêtes avec **impressions élevées** + **CTR < 2.5%**
   - requêtes avec position moyenne **4 à 20**.
4. Sélectionner top 10 opportunités.

### Tous les mardis (60 min)
1. Réécrire title + meta description de 5 pages ciblées.
2. Ajouter 1 bloc de lien interne vers `/contact` et 2 pages money.
3. Vérifier build/sitemap.

### Tous les jeudis (45 min)
1. Publier 1 nouvel article “ville + intention”.
2. Ajouter FAQ ciblée requêtes longues.
3. Lier vers page pays + `/contact`.

### Tous les vendredis (30 min)
1. Contrôle indexation (pages récentes).
2. Vérifier CTR delta sur pages retouchées.
3. Noter gagnants/perdants et ajuster semaine suivante.

## 7) Règles d’or éditoriales
- 1 mot-clé principal par page + 3 variantes sémantiques.
- H1 unique orienté intention business.
- FAQ courte (3 questions) pour longue traîne.
- 3 liens internes minimum vers pages money.
- CTA visible “Demander un devis” dans chaque contenu.
